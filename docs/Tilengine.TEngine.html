class function Singleton(Hres, Vres, numLayers, numSprites, numAnimations : Integer) : TEngine;

Initializes the graphic engine

This is a singleton object: calling Init multiple times will return the same reference

Parameters
hres
horizontal resolution in pixels
vres
vertical resolution in pixels
numLayers
number of layers
numSprites
number of sprites
numAnimations
number of animations
Returns

Engine instance

procedure SetBackgroundColor(const value : TColor); overload;

Sets the background color, that is the color of the pixel when there isn't any layer or sprite at that position

procedure SetBackgroundColor(const tilemap : TTilemap); overload;

Sets the background color from a tilemap, that is the color of the pixel when there isn't any layer or sprite at that position

procedure DisableBackgroundColor;

Disables background color rendering. If you know that the last background layer will always cover the entire screen, you can disable it to gain some performance

procedure SetRenderTarget(data : TArray<Byte>; pitch : Integer);

Sets the output surface for rendering

The render target pixel format must be 32 bits RGBA

Parameters
data
Array of bytes that will hold the render target
pitch
Number of bytes per each scanline of the framebuffer
procedure SetRasterCallback(callback : TVideocallback);

Enables raster effects processing, like a virtual HBLANK interrupt where any render parameter can be modified between scanlines.

Parameters
callback
name of the user-defined function to call for each scanline. Set Null to disable.
procedure SetFrameCallback(callback : TVideocallback);

Enables user callback for each drawn frame, like a virtual VBLANK interrupt

Parameters
callback
name of the user-defined function to call for each frame. Set Null to disable.
procedure SetCustomBlendFunction(customfunction : TBlendFunction);

Sets custom blend function to use in sprites or background layers when cref="Blend.Custom" mode is selected with the cref="Layer.BlendMode" and cref="Sprite.BlendMode" properties.

Parameters
function
user-defined function to call when blending that takes two integer arguments: source component intensity, destination component intensity, and returns the desired intensity.
procedure BeginFrame(frame : Integer);

Starts active rendering of the current frame

This method is used for active rendering combined with DrawNextScanline(), instead of using delegates for raster effects

Parameters
frame
Timestamp value
